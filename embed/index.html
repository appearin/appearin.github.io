The <whereby-embed> element

This is currently experimental.

You can try it out like this: <script type=module src="https://whereby.dev/embed/whereby-embed.js"></script>
<style>whereby-embed { height: 700px; }</style>
<whereby-embed embed subdomain=yourcompany room="/your_room"></whereby-embed>

You can also use a shortcut of giving the room attribute your full URL which is convenient when dealing with the API. This is also required when you are using the hostRoomUrl: <whereby-embed room="https://yourcompany.whereby.com/your_room?roomKey=3fe345a"></whereby-embed>

Attributes you can use

Attribute Value Description Usage
room The room you want to embed This field is required <whereby-embed room="/your_room" />
minimal Nothing (on) or "off" Apply minimal UI for embed scenarios <whereby-embed minimal />
displayName Any string Set displayname of participant <whereby-embed displayName="John Smith" />
audio=off "off" Enter meeting with audio off <whereby-embed audio=off />
video=off "off" Enter meeting with video off <whereby-embed video=off />
background=off "off" Render without background to let embedding app render its own <whereby-embed background=off />
chat Nothing (on) or "off" Enable/disable chat <whereby-embed chat=off /> <whereby-embed minimal chat />
people "off" Disable the participant list <whereby-embed people=off />
leaveButton Nothing (on) or "off" Enable/disable the leave button <whereby-embed leaveButton=off /> <whereby-embed minimal leaveButton />
screenshare Nothing (on) or "off" Enable/disable screenshare <whereby-embed screenshare=off /> <whereby-embed minimal screenshare />
floatSelf Nothing (on) or "off" Float the self view to the bottom right <whereby-embed floatSelf=off /> <whereby-embed minimal floatSelf />

Listening to events

You can listen for events on the <whereby-embed> element. The following events are supported:

Event type Description Properties (from "detail")
ready Basic dependencies has loaded –
knock The local user knocks to get into the room –
participantupdate A new participant join/leave { count: Number }
leave The local user leaves –

You can use standard Javascript to listen to the events. Here's a small demo: 

  const elm = document.querySelector("whereby-embed");
  const output = document.querySelector("output");
  const events = [];
  function logEvent(event) {
    output.innerText += `got event ${JSON.stringify({ type: event.type, detail: event.detail})}\n`;
  }
  elm.addEventListener("ready", logEvent)
  elm.addEventListener("participantupdate", logEvent)
  elm.addEventListener("leave", logEvent)
