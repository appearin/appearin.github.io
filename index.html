Developer API [Beta]

appear.in at its nature allows for conjuring rooms into existence whenever needed by allowing for any randomly generated string being a room name. We have started looking into what an API for appear.in might look like. We have yet to develop a full API, so the functionality available today should be considered experimental.

This API is provided as is, that is to say without any uptime or stability guarantees. The API is likely to change without much notice.

If you are planning to use appear.in in a live service, please contact us at api@appear.in.

Do consider subscribing to our API mailing list.

What can I do with the API?

  • Check for appear.in compatibility in user's browser (check for required WebRTC element)
  • Create a room from any webpage
  • Embed a room
