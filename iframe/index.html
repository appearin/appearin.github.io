basic <iframe> example

This shows simply how to use iframe to embed Whereby. <iframe style="width: 100%; height: 600px"
src="https://example.whereby.com/room?embed&iframeSource=example&displayName=Some%20name"
allow="camera; microphone; fullscreen; speaker; display-capture"></iframe>
